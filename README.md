# Smart Invoice Automation System

AI-powered invoice processing tool built using FastAPI and Python.

## Features
- Extracts Invoice Number
- Extracts Invoice Date
- Extracts Subtotal
- Extracts VAT
- Extracts Total Amount
- Handles multiple invoice formats
- Flexible regex-based parsing
- Professional SaaS-style UI

## Tech Stack
- Python
- FastAPI
- pdfplumber
- Regex-based text parsing
- Tailwind CSS UI

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run server:
   uvicorn main:app --reload

3. Open:
   http://127.0.0.1:8000
